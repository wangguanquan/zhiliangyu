# zhiliangyu
玉器展示网站（练习用）

## 模块说明

- zhiliangyu-service 提供画面和服务接口
- zhiliangyu-dalgen 提供数据库访问

## mybatis dalgen

mybatis delgen是一个maven插件，将项目中repository放入本地maven仓库中。
cd 到项目zhiliangyu-dalgen，然后执行mvn bree-mybatis:gen运行此插件
